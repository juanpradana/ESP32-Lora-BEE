# ESP32-Lora-BEE
This repository for ESP32 to communication eith other esp32 using Lora BEE.

# Requirement
SPI.h Library

LoRa.h Library

# Wiring
ESP32 >>  Lora BEE

GND   >> GND

3.3V  >> 3.3V

D19   >> MISO

D23   >> MOSI

D18   >> SCK

D5    >> NSS

D14   >> RESET

D2    >> DIO0
